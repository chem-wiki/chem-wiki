# 铝（Al）- 竞赛核心考点

### 1. 核心形态（酸碱介质）
- **Al(III)**：**Al³⁺**（无色）。典型的两性元素。
  - 酸性中：以水合离子 **[Al(H₂O)₆]³⁺** 存在，易水解显酸性。
  - 碱性中：加碱先生成白色胶状沉淀 **Al(OH)₃**，碱过量溶解为 **[Al(OH)₄]⁻**（四羟基合铝酸根，常简写为AlO₂⁻）。
  - 氨水中：生成 **Al(OH)₃** 沉淀，**不溶**于过量氨水（与Zn²⁺、Cu²⁺、Ag⁺等区分）。

### 2. 特征反应
- **两性反应**：
  - 溶于酸：`Al(OH)₃ + 3H⁺ = Al³⁺ + 3H₂O`
  - 溶于碱：`Al(OH)₃ + OH⁻ = [Al(OH)₄]⁻`
- **双水解反应**：
  Al³⁺与弱酸根（如S²⁻、CO₃²⁻、HCO₃⁻、AlO₂⁻）在水溶液中相遇，会发生彻底的双水解，生成Al(OH)₃沉淀和相应的气体。
  `Al³⁺ + 3HCO₃⁻ = Al(OH)₃↓ + 3CO₂↑`（泡沫灭火器原理）
  `Al³⁺ + 3S²⁻ + 6H₂O = 2Al(OH)₃↓ + 3H₂S↑`
- **铝热反应**：
  铝粉与金属氧化物（如Fe₂O₃、MnO₂、Cr₂O₃）在高温下发生剧烈反应，放出大量热，用于冶炼难熔金属或焊接钢轨。
  `2Al + Fe₂O₃ = Al₂O₃ + 2Fe`（高温）

### 3. 竞赛易错点
- **易错1**：认为Al能与纯水反应。实际上铝表面有致密的Al₂O₃氧化膜，阻止了反应。若去除氧化膜（如用Hg形成铝汞齐），铝能与水剧烈反应放出H₂。
- **易错2**：将偏铝酸根写为AlO₂⁻参与配平。在水溶液中，真实的形态是 **[Al(OH)₄]⁻**，竞赛中推荐使用此形态，能更清晰地体现配位化学本质。
- **易错3**：认为Al(OH)₃溶于氨水。**错误**，氨水碱性太弱，不足以使Al(OH)₃转化为[Al(OH)₄]⁻，这是分离Al³⁺与Zn²⁺/Cu²⁺的重要方法。
