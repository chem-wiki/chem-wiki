# 碱金属（Li, Na, K, Rb, Cs）- 竞赛核心考点

### 1. 核心形态（酸碱介质）
- **M(I)**：**M⁺**（无色）。碱金属在化合物中几乎总是+1价。
  - 极易溶于水，形成强碱（MOH）。
  - 碱性递变：LiOH < NaOH < KOH < RbOH < CsOH。
  - 溶解度异常：Li⁺的许多盐（如Li₂CO₃、LiF、Li₃PO₄）微溶，与Mg²⁺相似（对角线规则）。

### 2. 特征反应
- **与氧气的反应（产物随原子序数增大而变复杂）**：
  - Li：主要生成正常氧化物 `4Li + O₂ = 2Li₂O`。
  - Na：主要生成过氧化物 `2Na + O₂ = Na₂O₂`（淡黄色）。
  - K, Rb, Cs：主要生成超氧化物 `M + O₂ = MO₂`（如KO₂，橙黄色）。
- **液氨溶液（溶剂化电子）**：
  碱金属溶于液氨，生成深蓝色、导电性极强、具极强还原性的溶液。
  `M + (x+y)NH₃ ⇌ M(NH₃)x⁺ + e(NH₃)y⁻`（溶剂化电子呈蓝色）。
  久置或加催化剂（如Fe）会放出H₂：`2M + 2NH₃ = 2MNH₂ + H₂↑`。
- **特征沉淀鉴定**：
  - **Na⁺**：与醋酸铀酰锌生成黄绿色结晶沉淀 `NaZn(UO₂)₃(CH₃COO)₉·6H₂O`；或与焦锑酸钾生成白色沉淀 `NaSb(OH)₆`。
  - **K⁺**：与亚硝酸钴钠生成黄色沉淀 `K₂Na[Co(NO₂)₆]`；或与四苯硼钠生成白色沉淀 `K[B(C₆H₅)₄]`。

### 3. 竞赛易错点
- **易错1**：认为所有碱金属的碳酸盐都易溶且受热不分解。**错误**，Li₂CO₃微溶于水，且受热易分解（`Li₂CO₃ = Li₂O + CO₂↑`），这体现了Li的极化力强。
- **易错2**：忽略碱金属在液氨中的状态。深蓝色溶液的还原性来自溶剂化电子，而不是碱金属原子本身。
- **易错3**：焰色反应的干扰。观察K的焰色（紫色）时，必须透过蓝色钴玻璃，以滤去Na的黄色焰色干扰。
