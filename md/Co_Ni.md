# 钴（Co）与镍（Ni）- 竞赛核心考点

### 1. 核心形态（酸碱介质）
- **Co(II)**：**Co²⁺**（粉红色）。水溶液中最稳定的氧化态。
  - 碱性中：生成粉红色 **Co(OH)₂** 沉淀，在空气中缓慢氧化为棕黑色的 **Co(OH)₃**。
- $$ \ce{配合物：[Co(H₂O)₆]²⁺（粉红，八面体） <->> [CoCl₄]²⁻（蓝色，四面体）。} $$
- **Co(III)**：**Co³⁺**。简单离子在水中极不稳定（强氧化剂，能氧化水放出O₂），但**配合物极稳定**（如[Co(NH₃)₆]³⁺）。
- **Ni(II)**：**Ni²⁺**（亮绿色）。最稳定的氧化态。
  - 碱性中：生成浅绿色 **Ni(OH)₂** 沉淀，不溶于过量碱，溶于氨水生成蓝色 **[Ni(NH₃)₆]²⁺**。
- **Ni(IV) / Ni(III)**：极罕见，仅在强氧化剂（如NaClO、Br₂）碱性条件下存在，如黑色的NiO(OH)或NiO₂。

### 2. 特征反应
- **Co(II)的变色硅胶原理**：
  利用Co²⁺配位构型的转变。
  $$ \ce{[Co(H₂O)₆]²⁺（粉红） + 4Cl⁻ <->> [CoCl₄]²⁻（蓝） + 6H₂O} $$（加热或脱水促使平衡右移变蓝，吸水变粉红）。
- **Co(III)配合物的生成（氧化配位）**：
  在氨水和NH₄Cl存在下，Co²⁺极易被空气中的O₂或H₂O₂氧化为Co(III)配合物。
  $$ \ce{4Co²⁺ + 4NH₄⁺ + 20NH₃ + O₂ -> 4[Co(NH₃)₆]³⁺ + 2H₂O} $$（这是制备Co(III)配合物的经典方法）。
- **镍的特征鉴定（丁二酮肟/镍试剂）**：
  在弱碱性（氨水）条件下，Ni²⁺与丁二酮肟（HDmg）生成鲜艳的**鲜红色沉淀**。
  $$ \ce{Ni²⁺ + 2HDmg + 2NH₃ -> Ni(Dmg)₂↓（红） + 2NH₄⁺} $$（该配合物具有分子内氢键，平面正方形结构）。

### 3. 竞赛易错点
- **易错1**：认为Co³⁺在水溶液中稳定。**错误**，Co³⁺的氧化性极强（E° = 1.81V），会迅速氧化水。只有形成配合物（如与NH₃、CN⁻、NO₂⁻配位）后，电极电势大幅降低，Co(III)才变得极稳定。
- **易错2**：混淆Co(OH)₂和Ni(OH)₂的氧化难易。
  - Co(OH)₂在空气中即可被O₂氧化为Co(OH)₃。
  - Ni(OH)₂在空气中稳定，必须用强氧化剂（如NaClO、Br₂水）才能氧化为高价镍氧化物。
- **易错3**：忽略[Ni(CO)₄]的剧毒性和挥发性。四羰基镍是无色液体，沸点低（43℃），剧毒，加热至200℃分解为Ni和CO（用于蒙德法提纯镍）。
