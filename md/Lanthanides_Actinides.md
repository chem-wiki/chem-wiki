# 镧系与锕系（Lanthanides & Actinides）- 竞赛核心考点

### 1. 核心形态（酸碱介质）
- **Ln(III)**：**Ln³⁺**。镧系元素最稳定、最普遍的氧化态。
  - 颜色：许多Ln³⁺具有特征颜色（由f-f跃迁引起，吸收带极窄，颜色不受配体影响）。
  - 碱性中：生成难溶的 **Ln(OH)₃** 沉淀（碱性随原子序数增加而减弱，即La(OH)₃最强，Lu(OH)₃最弱，镧系收缩效应）。
- **异常价态（+2 和 +4）**：
  - **Ce(IV)**：**Ce⁴⁺**（橙黄色）。强氧化剂，常用于氧化还原滴定（铈量法）。
  - **Eu(II)**：**Eu²⁺**（黄绿色）。强还原剂，易被氧化为Eu³⁺。
  - **Sm(II)**：**Sm²⁺**（血红色）。强还原剂。
- **锕系（以铀U为例）**：
  - **U(VI)**：**UO₂²⁺**（双氧铀离子，黄色）。最稳定的形态，在水溶液中常发绿色荧光。
  - **U(IV)**：**U⁴⁺**（绿色）。易被氧化为UO₂²⁺。

### 2. 特征反应
- **铈量法（Ce⁴⁺的氧化性）**：
  在酸性（通常为H₂SO₄）条件下，Ce⁴⁺是优良的滴定剂，比KMnO₄更稳定，且无副反应。
  `Ce⁴⁺ + Fe²⁺ = Ce³⁺ + Fe³⁺`
- **镧系元素的沉淀分离**：
  - 草酸盐：`2Ln³⁺ + 3C₂O₄²⁻ = Ln₂(C₂O₄)₃↓`（难溶于弱酸，用于从矿石中沉淀分离镧系）。
  - 氟化物：`Ln³⁺ + 3F⁻ = LnF₃↓`（极难溶）。
- **铀的特征鉴定**：
  UO₂²⁺与[Fe(CN)₆]⁴⁻（黄血盐）反应生成红棕色沉淀。
  `2UO₂²⁺ + [Fe(CN)₆]⁴⁻ = (UO₂)₂[Fe(CN)₆]↓`

### 3. 竞赛易错点
- **易错1**：认为镧系元素的化学性质差异很大。**错误**，由于镧系收缩，Ln³⁺的半径极其接近，导致它们的化学性质极其相似，分离极其困难（通常采用离子交换法或溶剂萃取法）。
- **易错2**：忽略Ce⁴⁺在不同酸中的电极电势差异。Ce⁴⁺的氧化性在HClO₄中极强（1.70V），在HNO₃中次之（1.61V），在H₂SO₄中较弱（1.44V），这是因为形成了不同稳定性的配合物。
- **易错3**：将UO₂²⁺中的U化合价算错。UO₂²⁺中U的化合价为+6价，它是直线型离子，氧原子通过双键与铀结合。
