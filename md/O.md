# 氧（O）与过氧化物 - 竞赛核心考点

### 1. 核心形态（酸碱介质）
- **O(-I)**：**H₂O₂ / O₂²⁻**（过氧化物）。
  - 既有氧化性又有还原性，但以氧化性为主。
  - 酸性中：强氧化剂（如氧化Fe²⁺、I⁻），还原产物为H₂O。
  - 碱性中：氧化性减弱，还原性增强（如被MnO₄⁻、Cl₂氧化），氧化产物为O₂。
- **O(-1/2)**：**O₂⁻**（超氧化物，如KO₂）。
  - 极强的氧化剂和供氧剂，遇水或CO₂剧烈反应放出O₂。
- **O(0)**：**O₂ / O₃**。
  - 臭氧（O₃）是极强的氧化剂（强于O₂），能氧化Ag、Hg、S²⁻等，且能裂解碳碳双键（臭氧化反应）。

### 2. 特征反应
- **H₂O₂的氧化还原两性**：
  - 氧化性（遇强还原剂）：`H₂O₂ + 2I⁻ + 2H⁺ = I₂ + 2H₂O`
  - 还原性（遇强氧化剂）：`5H₂O₂ + 2MnO₄⁻ + 6H⁺ = 5O₂↑ + 2Mn²⁺ + 8H₂O`（常用于定量滴定H₂O₂）。
- **超氧化钾作为供氧剂（航天/潜水）**：
  `4KO₂ + 2CO₂ = 2K₂CO₃ + 3O₂`
  `4KO₂ + 2H₂O = 4KOH + 3O₂`
- **臭氧的鉴定（碘化钾淀粉试纸）**：
  O₃能将中性或弱酸性溶液中的I⁻氧化为I₂，使淀粉变蓝。
  `O₃ + 2I⁻ + H₂O = O₂ + I₂ + 2OH⁻`（注意：O₂不能在无酸条件下迅速氧化I⁻，这是区分O₃和O₂的方法）。

### 3. 竞赛易错点
- **易错1**：认为H₂O₂的分解是放热反应，所以高温下更稳定。**错误**，H₂O₂分解（`2H₂O₂ = 2H₂O + O₂`）是放热且熵增的自发反应，受热、光照或催化剂（如MnO₂、Fe³⁺、过氧化氢酶）作用下极易爆炸性分解。
- **易错2**：混淆过氧化物和二氧化物。如BaO₂是过氧化钡（含O₂²⁻，遇酸生成H₂O₂），而MnO₂、PbO₂是二氧化物（含O²⁻，遇酸不生成H₂O₂，而是表现强氧化性放出Cl₂或O₂）。
- **易错3**：配平H₂O₂作还原剂的方程式时，氧气来源写错。H₂O₂作还原剂时，生成的O₂**全部**来自于H₂O₂中的过氧键断裂。
