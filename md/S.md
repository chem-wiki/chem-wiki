# 硫（S）- 竞赛核心考点

### 1. 核心形态（酸碱介质）
- **S(VI)**：**SO₄²⁻**。极稳定，但在浓硫酸加热条件下具有强氧化性。
- **S(IV)**：**SO₂ / H₂SO₃ / SO₃²⁻**。
  - 酸性中：主要以溶解的 **SO₂** 或 **H₂SO₃** 形式存在，具较强还原性，也有弱氧化性（如与H₂S反应）。
  - 碱性中：主要为 **SO₃²⁻**，还原性显著增强，极易被空气中的O₂氧化为SO₄²⁻。
- **S(II)**：**S₂O₃²⁻**（硫代硫酸根）。在碱性或中性中稳定，在酸性中极易歧化分解为S和SO₂。是极重要的配体（如[Ag(S₂O₃)₂]³⁻）和还原剂（用于碘量法）。
- **S(-II)**：**H₂S / HS⁻ / S²⁻**。
  - 酸性中：**H₂S** 气体，具强还原性。
  - 碱性中：**S²⁻**，极易被氧化，且能与绝大多数重金属离子生成极难溶的硫化物沉淀。

### 2. 特征反应
- **硫代硫酸根的酸性歧化**：
  `S₂O₃²⁻ + 2H⁺ = S↓（淡黄色） + SO₂↑ + H₂O`（用于鉴别S₂O₃²⁻与SO₃²⁻）。
- **碘量法核心反应**：
  `I₂ + 2S₂O₃²⁻ = 2I⁻ + S₄O₆²⁻`（连四硫酸根）。注意：若用强氧化剂（如Cl₂、KMnO₄），则S₂O₃²⁻会被直接氧化为SO₄²⁻。
- **归中反应（制备单质硫）**：
  `2H₂S + SO₂ = 3S↓ + 2H₂O`（常温下即可发生，体现H₂S的还原性和SO₂的氧化性）。
- **多硫化物的生成**：
  `S²⁻ + (n-1)S = Sₙ²⁻`（多硫离子，呈黄色至橙红色，酸性下不稳定，分解放出H₂S并析出S）。

### 3. 竞赛易错点
- **易错1**：认为S₂O₃²⁻与Ba²⁺会产生沉淀。实际上BaS₂O₃是可溶的（或微溶，但不像BaSO₄那样难溶），常用于分离SO₄²⁻和S₂O₃²⁻。
- **易错2**：忽略S²⁻的水解。Na₂S溶液显强碱性（`S²⁻ + H₂O ⇌ HS⁻ + OH⁻`），与Al³⁺、Fe³⁺等相遇会发生双水解（如生成Al(OH)₃和H₂S），而不是生成Al₂S₃沉淀。
- **易错3**：混淆连二亚硫酸根（S₂O₄²⁻，保险粉，强还原剂）与连四硫酸根（S₄O₆²⁻，碘量法产物）的结构与性质。
