# 锌（Zn）与镉（Cd）、汞（Hg）- 竞赛核心考点

### 1. 核心形态（酸碱介质）
- **Zn(II)**：**Zn²⁺**（无色）。两性元素。
  - 碱性中：先生成白色 **Zn(OH)₂** 沉淀，碱过量溶解为 **[Zn(OH)₄]²⁻**（锌酸根）。
  - 氨水中：Zn(OH)₂溶于氨水生成 **[Zn(NH₃)₄]²⁺**。
- **Cd(II)**：**Cd²⁺**（无色）。非两性（或极弱两性）。
  - 碱性中：生成白色 **Cd(OH)₂** 沉淀，不溶于过量碱。
  - 氨水中：溶于氨水生成 **[Cd(NH₃)₄]²⁺**。
- **Hg(II)**：**Hg²⁺**（无色）。
  - 碱性中：生成黄色 **HgO** 沉淀（无Hg(OH)₂）。
  - 与I⁻：先生成红色 **HgI₂** 沉淀，I⁻过量溶解为无色的 **[HgI₄]²⁻**（奈斯勒试剂成分）。
- **Hg(I)**：**Hg₂²⁺**（亚汞离子，存在金属-金属键）。
  - 遇碱：歧化为Hg和HgO。

### 2. 特征反应
- **Hg(I)与Hg(II)的转化（歧化与归中）**：
  $$ \ce{Hg₂²⁺ <->> Hg + Hg²⁺} $$。加入能与Hg²⁺形成难溶物（如S²⁻、OH⁻）或配合物（如I⁻、CN⁻）的试剂，会促使Hg₂²⁺歧化。
  例如加I⁻：$$ \ce{Hg₂²⁺ + 4I⁻ -> Hg↓ + [HgI₄]²⁻} $$。
- **奈斯勒试剂（K₂[HgI₄]的碱性溶液）鉴定NH₄⁺**：
  生成红棕色沉淀（碘化氨基氧汞）。
  $$ \ce{2[HgI₄]²⁻ + NH₄⁺ + 4OH⁻ -> HgO·Hg(NH₂)I↓ + 7I⁻ + 3H₂O} $$
- **硫化物的颜色**：
  - **ZnS**：白色（唯一白色的硫化物沉淀）。
  - **CdS**：黄色（镉黄，重要颜料）。
  - **HgS**：黑色（辰砂/朱砂为红色，沉淀通常为黑色，极难溶，仅溶于王水或浓Na₂S溶液）。

### 3. 竞赛易错点
- **易错1**：认为Cd(OH)₂是两性的。**错误**，Cd(OH)₂不溶于过量NaOH，这是分离Zn和Cd的重要方法。
- **易错2**：写亚汞离子为Hg⁺。**错误**，必须写为双聚离子 **Hg₂²⁺**。
- **易错3**：忽略Hg₂Cl₂（甘汞）遇氨水的歧化反应。$$ \ce{Hg₂Cl₂ + 2NH₃ -> Hg↓（黑） + Hg(NH₂)Cl↓（白） + NH₄⁺ + Cl⁻} $$（沉淀变黑，用于鉴定亚汞）。
