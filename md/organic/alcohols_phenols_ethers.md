# 有机化学：醇、酚和醚

## 1. 醇 ($R-OH$)
- **结构**：羟基连接在 $sp^3$ 碳上。
- **物理性质**：由于分子间氢键，沸点较高。水溶性（低级醇）。
- **酸性**：弱酸 ($pK_a \approx 16-18$)。$1^\circ > 2^\circ > 3^\circ$（烷基通过诱导效应使醇盐离子不稳定）。

### 合成
1.  **卤代烃的水解**：$R-X + NaOH(aq) \rightarrow R-OH$。
2.  **烯烃的水合**：酸催化或硼氢化-氧化。
3.  **羰基化合物的还原**：
    - 醛/酮 $\xrightarrow{NaBH_4} 1^\circ/2^\circ$ 醇。
    - 酯/酸 $\xrightarrow{LiAlH_4} 1^\circ$ 醇。
4.  **格氏反应**：$RMgX$ 进攻羰基 $\rightarrow$ 醇。（甲醛 $\rightarrow 1^\circ$，醛 $\rightarrow 2^\circ$，酮 $\rightarrow 3^\circ$）。

### 反应
1.  **脱水 ($E1/E2$)**：酸催化 ($H_2SO_4, \Delta$) 生成烯烃（主要为扎伊采夫产物）。
2.  **氧化**：
    - $1^\circ \xrightarrow{PCC} \text{醛} \xrightarrow{KMnO_4/Jones} \text{羧酸}$。
    - $2^\circ \xrightarrow{Cr(VI)} \text{酮}$。
    - $3^\circ$：不反应（无 $\alpha$-氢）。
3.  **取代 ($S_N1/S_N2$)**：
    - 与 $HX$：卢卡斯试剂 ($ZnCl_2/HCl$) 鉴别 $1^\circ$（慢）、$2^\circ$、$3^\circ$（快，立即浑浊）。
    - 与 $SOCl_2$ 或 $PBr_3$：转化为卤代烃，构型翻转 ($S_N2$)。
4.  **酯化**：与羧酸（费歇尔）或酰氯反应生成酯。

## 2. 酚 ($Ar-OH$)
- **酸性**：比醇强 ($pK_a \approx 10$)，因为苯氧负离子 ($Ar-O^-$) 具有共振稳定性。
    - 吸电子基团（如 $-NO_2$）显著增加酸性（苦味酸）。

### 反应
1.  **亲电芳香取代**：$-OH$ 是强活化基团（邻/对位定位）。
    - 溴化：$Br_2/H_2O$ 生成 2,4,6-三溴苯酚（白色沉淀）。
    - 硝化：稀 $HNO_3$ 生成邻/对-硝基苯酚。
2.  **瑞穆尔-蒂曼反应**：苯酚 + $CHCl_3/NaOH \rightarrow$ 水杨醛（邻-羟基苯甲醛）。
3.  **科尔贝-施密特反应**：苯酚盐 + $CO_2 \xrightarrow{\Delta, P} \text{水杨酸}$。

## 3. 醚 ($R-O-R'$)
- **结构**：氧桥。相对惰性（良溶剂）。
- **合成（威廉姆逊醚合成）**：
    - $S_N2$ 反应：$R-O^- Na^+ + R'-X \rightarrow R-O-R' + NaX$。
    - *限制*：$R'-X$ 必须是 $1^\circ$ 以避免消除。

### 反应
1.  **强酸裂解 ($HI/HBr$)**：
    - 氧的质子化使其成为良好的离去基团。
    - $S_N2$（对于 $1^\circ$）：亲核试剂 ($I^-$) 进攻位阻较小的碳。
    - $S_N1$（对于 $3^\circ$）：首先形成稳定的碳正离子。
    - 产物：2 个卤代烃（酸过量）或醇 + 卤代烃。

## 4. 环氧化合物（环氧乙烷）
- **开环**：张力大的三元环反应性高。
    - **酸催化**：亲核试剂进攻**取代较多**的碳（类 $S_N1$ 特性）。构型翻转。
    - **碱催化**：亲核试剂进攻**取代较少**的碳（$S_N2$，立体位阻控制）。构型翻转。
