# 有机化学：芳香族化合物

## 1. 苯的结构
- **分子式**：$C_6H_6$（凯库勒结构式）。
- **结构**：平面六边形环，具有相等的 C-C 键长（$1.40 \mathring{A}$）。介于单键（$1.54 \mathring{A}$）和双键（$1.34 \mathring{A}$）之间。
- **共振**：通过离域 $\pi$ 电子（环状共轭）稳定。共振能 $\approx 36 \text{ kcal/mol}$。

## 2. 芳香性（休克尔规则）
一个化合物是芳香族的，如果它满足：
1.  **环状**：具有重叠 p 轨道的连续环。
2.  **平面**：环上的所有原子位于同一平面（$sp^2$）。
3.  **共轭**：交替的双键/单键或 p 轨道中的孤对电子。
4.  **休克尔规则**：具有 **$4n + 2$** 个 $\pi$ 电子（$n = 0, 1, 2...$）。
    - *反芳香性*：满足 1-3 但具有 $4n$ 个 $\pi$ 电子（不稳定）。
    - *非芳香性*：不满足 1、2 或 3。

## 3. 亲电芳香取代 (EAS)
苯发生取代反应而不是加成反应，以保持芳香稳定性。
**机理**：
1.  **亲电试剂 ($E^+$) 的生成**：需要路易斯酸催化剂（如 $AlCl_3, FeBr_3$）。
2.  **进攻**：$\pi$ 电子进攻 $E^+$，形成共振稳定的碳正离子（$\sigma$-络合物 / 苯铵离子）。此步骤是决速步（芳香性丧失）。
3.  **重新芳构化**：碱从 $sp^3$ 碳上夺取质子 ($H^+$)，恢复芳香性。

### 主要 EAS 反应
| 反应 | 试剂 | 亲电试剂 ($E^+$) | 产物 |
| :--- | :--- | :--- | :--- |
| **卤代** | $Cl_2/FeCl_3$ 或 $Br_2/FeBr_3$ | $Cl^+$ 或 $Br^+$ | 氯苯 / 溴苯 |
| **硝化** | $HNO_3/H_2SO_4$ | $NO_2^+$ (硝鎓离子) | 硝基苯 |
| **磺化** | 发烟 $H_2SO_4$ ($SO_3$) | $HSO_3^+$ 或 $SO_3$ | 苯磺酸 (可逆) |
| **傅-克烷基化** | $R-Cl/AlCl_3$ | $R^+$ (碳正离子) | 烷基苯 (注意重排/多烷基化) |
| **傅-克酰基化** | $RCOCl/AlCl_3$ | $R-C \equiv O^+$ (酰基阳离子) | 芳基酮 (无重排) |

## 4. 取代基效应
环上已有的取代基指导进入的亲电试剂并影响反应活性。

### 活化基团（给电子基团 - EDG）
- 增加环上的电子密度（亲核性）。
- 使环比苯**更活泼**。
- 指导进入的基团到**邻/对位 (Ortho/Para)**。
    - *强*：$-OH, -NH_2, -OR$（共振给电子）。
    - *弱*：$-CH_3, -R$（诱导/超共轭）。

### 钝化基团（吸电子基团 - EWG）
- 降低电子密度。
- 使环**较不活泼**。
- 指导进入的基团到**间位 (Meta)**。
    - *强*：$-NO_2, -CN, -SO_3H, -COOH, -CHO$（共振吸电子）。
    - *弱*：$-F, -Cl, -Br, -I$（**例外**：钝化但**邻/对位**定位，因为孤对电子共振 vs 强诱导吸电子）。
