# 稀有气体（Noble Gases）- 竞赛核心考点

### 1. 核心形态（酸碱介质）
- **Xe(II)**：**XeF₂**。直线型分子，强氧化剂。
  - 在水中缓慢水解：`2XeF₂ + 2H₂O = 2Xe↑ + O₂↑ + 4HF`。
- **Xe(IV)**：**XeF₄**。平面正方形分子。
  - 遇水剧烈歧化：`6XeF₄ + 12H₂O = 2XeO₃ + 4Xe↑ + 3O₂↑ + 24HF`。
- **Xe(VI)**：**XeF₆ / XeO₃ / HXeO₄⁻**。
  - XeF₆：变形八面体，极易水解生成XeO₃。
  - XeO₃：无色晶体，极易爆炸，水溶液（氙酸）具强氧化性。
  - 碱性中：生成氙酸根（HXeO₄⁻），缓慢歧化为Xe和高氙酸根（XeO₆⁴⁻）。
- **Xe(VIII)**：**XeO₄ / XeO₆⁴⁻**。
  - 高氙酸盐（如Na₄XeO₆）是极强的氧化剂（能将Mn²⁺氧化为MnO₄⁻）。

### 2. 特征反应
- **氙氟化物的合成**：
  利用不同比例的Xe和F₂在加热或光照下反应。
  `Xe + F₂ = XeF₂`（Xe过量）
  `Xe + 2F₂ = XeF₄`（1:5比例，加压）
  `Xe + 3F₂ = XeF₆`（F₂大过量，高压）
- **XeF₆的彻底水解**：
  `XeF₆ + 3H₂O = XeO₃ + 6HF`（生成危险的爆炸物XeO₃）。
- **高氙酸盐的生成（歧化反应）**：
  在强碱性条件下，氙酸根发生歧化。
  `2HXeO₄⁻ + 2OH⁻ = XeO₆⁴⁻ + Xe↑ + O₂↑ + 2H₂O`

### 3. 竞赛易错点
- **易错1**：认为稀有气体绝对不反应。**错误**，自1962年合成出第一种稀有气体化合物（XePtF₆）以来，已发现大量Kr、Xe、Rn的化合物，主要与电负性极强的F和O结合。
- **易错2**：混淆XeF₂、XeF₄、XeF₆的水解产物。
  - XeF₂水解生成Xe和O₂。
  - XeF₄水解发生歧化，生成Xe和XeO₃。
  - XeF₆水解不发生氧化还原，直接生成XeO₃。
- **易错3**：忽略XeF₂的氟化剂和氧化剂双重身份。它不仅能提供F原子，还能将许多低价化合物氧化（如将BrO₃⁻氧化为BrO₄⁻）。
