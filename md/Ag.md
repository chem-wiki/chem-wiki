# 银（Ag）- 竞赛核心考点

### 1. 核心形态（酸碱介质）
- **Ag(I)**：**Ag⁺**（无色）。最常见且最稳定的氧化态。
  - 碱性中：加碱生成暗棕色的 **Ag₂O** 沉淀（AgOH极不稳定，室温下即分解）。
  - 氨水中：Ag₂O溶于氨水生成无色的 **[Ag(NH₃)₂]⁺**（银氨络离子，即托伦试剂）。
- **Ag(II) / Ag(III)**：极罕见，仅在强氧化剂（如S₂O₈²⁻、O₃）存在下，或形成特殊配合物时才能存在，是极强的氧化剂。

### 2. 特征反应
- **卤化银沉淀的生成与溶解**：
  - **AgCl**（白色）：溶于稀氨水（生成[Ag(NH₃)₂]⁺）。
  - **AgBr**（淡黄色）：微溶于浓氨水，溶于Na₂S₂O₃（生成[Ag(S₂O₃)₂]³⁻）。
  - **AgI**（黄色）：不溶于氨水，微溶于Na₂S₂O₃，溶于KCN（生成[Ag(CN)₂]⁻）。
- **银镜反应**：
  在碱性条件下，[Ag(NH₃)₂]⁺被含有醛基的化合物还原为金属银。
  $$ \ce{2[Ag(NH₃)₂]⁺ + RCHO + 3OH⁻ -> 2Ag↓ + RCOO⁻ + 4NH₃ + 2H₂O} $$
- **感光性**：
  卤化银（尤其是AgBr）见光分解：$$ \ce{2AgBr -> 2Ag + Br₂} $$（摄影底片原理）。

### 3. 竞赛易错点
- **易错1**：写Ag⁺与OH⁻反应生成AgOH沉淀。**错误**，应直接写生成Ag₂O和H₂O（$$ \ce{2Ag⁺ + 2OH⁻ -> Ag₂O↓ + H₂O} $$）。
- **易错2**：认为所有卤化银都溶于氨水。**错误**，AgI极难溶，其溶度积太小，氨水无法使其溶解，必须用更强的配体如CN⁻或S₂O₃²⁻。
- **易错3**：配制托伦试剂时氨水过量太多。氨水过量会降低试剂的灵敏度，且久置易生成易爆的雷酸银（AgONC）或氮化银（Ag₃N）。
