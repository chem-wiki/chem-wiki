# 如何贡献 (How to Contribute)

欢迎来到 **Chem Wiki**！这是一个致力于化学知识分享的开源项目。

## 项目地址
本项目托管于：[chem-wiki.llumi.org](https://chem-wiki.llumi.org)

## 我们欢迎您的贡献
无论是修复拼写错误、修正科学概念，还是添加全新的化学主题，我们都非常欢迎！您的每一份贡献都能帮助更多人更好地学习化学。

## 如何参与 (How to Fork/Clone/PR)

1.  **Fork 本仓库**：点击 GitHub 页面右上角的 "Fork" 按钮，将项目复制到您的账号下。
2.  **Clone 到本地**：
    ```bash
    git clone https://github.com/YOUR_USERNAME/chem-wiki.git
    ```
3.  **创建新分支**：在进行修改前，请创建一个新的分支。
    ```bash
    git checkout -b my-new-feature
    ```
4.  **提交更改**：完成修改后，提交您的代码。
    ```bash
    git commit -m "添加了关于XXX的内容"
    ```
5.  **推送到远程仓库**：
    ```bash
    git push origin my-new-feature
    ```
6.  **提交 Pull Request (PR)**：回到 GitHub 页面，点击 "New Pull Request" 按钮，描述您的更改并提交。

我们会尽快审核您的 PR。感谢您的参与！
