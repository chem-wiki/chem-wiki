# 碱土金属（Be, Mg, Ca, Sr, Ba）- 竞赛核心考点

### 1. 核心形态（酸碱介质）
- **M(II)**：**M²⁺**（无色）。
  - 碱性递变：Be(OH)₂（两性） < Mg(OH)₂（中强碱） < Ca(OH)₂ < Sr(OH)₂ < Ba(OH)₂（强碱）。
  - 溶解度规律：
    - 氢氧化物：自上而下溶解度**增大**（Mg(OH)₂难溶，Ba(OH)₂易溶）。
    - 硫酸盐/铬酸盐：自上而下溶解度**减小**（MgSO₄易溶，BaSO₄极难溶）。
    - 碳酸盐：均难溶。

### 2. 特征反应
- **铍的两性（对角线规则，与Al相似）**：
  - 溶于酸：`Be(OH)₂ + 2H⁺ = Be²⁺ + 2H₂O`
  - 溶于碱：`Be(OH)₂ + 2OH⁻ = [Be(OH)₄]²⁻`（铍酸根）。
- **镁的特征反应**：
  - 与水反应：冷水极慢，沸水较快 `Mg + 2H₂O(沸) = Mg(OH)₂ + H₂↑`。
  - 镁试剂（对硝基苯偶氮间苯二酚）鉴定：在碱性条件下，Mg(OH)₂沉淀吸附镁试剂呈现天蓝色。
- **钙、锶、钡的沉淀分离**：
  利用铬酸盐（CrO₄²⁻）的溶解度差异分离：
  - BaCrO₄：难溶于醋酸。
  - SrCrO₄：溶于醋酸，难溶于铬酸。
  - CaCrO₄：易溶。

### 3. 竞赛易错点
- **易错1**：认为Mg能像Na一样保存在煤油中。**错误**，Mg在空气中表面会形成致密的氧化膜，不需要保存在煤油中。
- **易错2**：忽略Be的剧毒性及其化合物的共价性。BeCl₂是共价化合物，熔融状态不导电，气态为直线型分子，固态为聚合物链。
- **易错3**：混淆硬水软化的原理。暂时硬水（含HCO₃⁻）加热即可软化（`Ca(HCO₃)₂ = CaCO₃↓ + CO₂↑ + H₂O`）；永久硬水（含SO₄²⁻、Cl⁻）需加Na₂CO₃或使用离子交换树脂。
