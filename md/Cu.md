# 铜（Cu）- 竞赛核心考点

### 1. 核心形态（酸碱介质）
- **Cu(II)**：**Cu²⁺**（蓝色）。最稳定的氧化态。
  - 碱性中：生成蓝色絮状沉淀 **Cu(OH)₂**，加热分解为黑色 **CuO**。
  - 氨水中：先生成Cu(OH)₂沉淀，氨水过量则溶解生成深蓝色的 **[Cu(NH₃)₄]²⁺**（铜氨络离子）。
- **Cu(I)**：**Cu⁺**（无色）。在水溶液中极易歧化为Cu和Cu²⁺，但可通过形成难溶物（如CuCl、CuI、Cu₂O）或配合物（如[Cu(CN)₄]³⁻）稳定存在。

### 2. 特征反应
- **Cu(I)的歧化与稳定**：
  `2Cu⁺ ⇌ Cu + Cu²⁺`（水溶液中自发进行）。
  若加入I⁻：`2Cu²⁺ + 4I⁻ = 2CuI↓（白色） + I₂`（Cu²⁺氧化I⁻，生成难溶的CuI使Cu(I)稳定，常用于碘量法测铜）。
- **斐林试剂/班氏试剂反应**：
  在碱性条件下，Cu²⁺（通常以酒石酸或柠檬酸配合物形式存在）被还原性糖（如葡萄糖）还原为砖红色的 **Cu₂O** 沉淀。
  `2Cu²⁺ + RCHO + 5OH⁻ = Cu₂O↓ + RCOO⁻ + 3H₂O`
- **铜氨配合物的生成**：
  `Cu²⁺ + 4NH₃ = [Cu(NH₃)₄]²⁺`（深蓝色溶液，特征鉴别反应）。

### 3. 竞赛易错点
- **易错1**：认为Cu²⁺与I⁻反应生成CuI₂。**错误**，Cu²⁺有氧化性，I⁻有还原性，两者会发生氧化还原反应生成CuI和I₂。
- **易错2**：忽略Cu⁺的歧化倾向。在没有沉淀剂或配位剂存在时，Cu⁺在水溶液中不能稳定存在。
- **易错3**：混淆Cu(OH)₂和碱式碳酸铜（Cu₂(OH)₂CO₃，铜绿）的生成条件。在空气中长期放置生成铜绿，加碱直接生成Cu(OH)₂。
