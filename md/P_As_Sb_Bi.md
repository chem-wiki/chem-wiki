# 磷（P）与砷（As）、锑（Sb）、铋（Bi）- 竞赛核心考点

### 1. 核心形态（酸碱介质）
- **P(V)**：**H₃PO₄ / PO₄³⁻**。正磷酸是中强酸，极稳定，无氧化性。
  - 焦磷酸（H₄P₂O₇）和偏磷酸（HPO₃）是多聚磷酸，能与蛋白质等大分子作用。
- **P(III)**：**H₃PO₃ / HPO₃²⁻**。亚磷酸是**二元酸**（含一个P-H键），具强还原性。
- **P(I)**：**H₃PO₂ / H₂PO₂⁻**。次磷酸是**一元酸**（含两个P-H键），具极强还原性（常用于化学镀镍）。
- **As(V) / Sb(V) / Bi(V)**：
  - 砷酸（H₃AsO₄）具中等氧化性。
  - 锑酸（HSb(OH)₆）微溶，其钠盐NaSb(OH)₆是鉴定Na⁺的特效试剂。
  - 铋酸盐（如NaBiO₃）是极强的氧化剂（能氧化Mn²⁺为MnO₄⁻）。
- **As(III) / Sb(III) / Bi(III)**：
  - 亚砷酸（H₃AsO₃）具两性，还原性较强。
  - Sb(III)和Bi(III)极易水解，生成难溶的碱式盐（如SbOCl、BiOCl）。

### 2. 特征反应
- **磷酸根的鉴定（钼酸铵试剂）**：
  在硝酸酸性条件下，PO₄³⁻与钼酸铵反应生成黄色的磷钼酸铵沉淀。
  `PO₄³⁻ + 12MoO₄²⁻ + 3NH₄⁺ + 24H⁺ = (NH₄)₃[P(Mo₁₂O₄₀)]↓ + 12H₂O`
- **砷的马氏试砷法（鉴定微量砷）**：
  在酸性条件下，锌粒将含砷化合物还原为剧毒的胂（AsH₃）气体，加热分解生成黑色的砷镜。
  `As₂O₃ + 6Zn + 12H⁺ = 2AsH₃↑ + 6Zn²⁺ + 3H₂O`
  `2AsH₃ = 2As↓（黑） + 3H₂↑`（加热）
- **Sb(III)和Bi(III)的水解**：
  将SbCl₃或BiCl₃溶液加水稀释，立即产生白色浑浊（加酸可抑制）。
  `Sb³⁺ + Cl⁻ + H₂O ⇌ SbOCl↓（白） + 2H⁺`
  `Bi³⁺ + Cl⁻ + H₂O ⇌ BiOCl↓（白） + 2H⁺`

### 3. 竞赛易错点
- **易错1**：认为H₃PO₃是三元酸。**绝对错误**，亚磷酸结构中有一个氢原子直接连在磷原子上（P-H键），不能电离，因此是二元酸，其正盐为Na₂HPO₃。
- **易错2**：混淆As(III)和As(V)与H₂S的反应。
  - As(III)在酸性中迅速生成黄色的 **As₂S₃** 沉淀。
  - As(V)在酸性中反应极慢，需先被H₂S还原为As(III)，再生成As₂S₃和S的混合沉淀。
- **易错3**：忽略Bi(III)的还原性。Bi(III)是最稳定的氧化态（惰性电子对效应），极难被氧化，只有在强碱性条件下用强氧化剂（如Cl₂）才能氧化为Bi(V)（如NaBiO₃）。
