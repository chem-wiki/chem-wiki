# 氮（N）- 竞赛核心考点

### 1. 核心形态（酸碱介质）
- **N(V)**：**HNO₃** / **NO₃⁻**。硝酸具强氧化性，浓度不同还原产物不同（浓硝酸主要还原为NO₂，稀硝酸主要还原为NO，极稀硝酸与活泼金属反应可还原为NH₄⁺）。
- **N(III)**：**HNO₂** / **NO₂⁻**。亚硝酸极不稳定，易歧化；NO₂⁻既有氧化性又有还原性，且是优良的配体（可形成硝基配合物或亚硝酸根配合物）。
- **N(-III)**：**NH₃** / **NH₄⁺**。氨是弱碱，也是极重要的配体（如[Cu(NH₃)₄]²⁺, [Ag(NH₃)₂]⁺）。

### 2. 特征反应
- **亚硝酸的歧化**：
  `3HNO₂ = HNO₃ + 2NO↑ + H₂O`（微热即可发生）
- **NO₂⁻的氧化还原两性**：
  - 遇强氧化剂（如KMnO₄）：`5NO₂⁻ + 2MnO₄⁻ + 6H⁺ = 5NO₃⁻ + 2Mn²⁺ + 3H₂O`
  - 遇强还原剂（如I⁻）：`2NO₂⁻ + 2I⁻ + 4H⁺ = 2NO↑ + I₂ + 2H₂O`
- **棕色环实验（鉴定NO₃⁻）**：
  在NO₃⁻溶液中加FeSO₄，沿管壁缓缓加入浓H₂SO₄。在界面处形成棕色环 **[Fe(H₂O)₅(NO)]²⁺**。
  `3Fe²⁺ + NO₃⁻ + 4H⁺ = 3Fe³⁺ + NO↑ + 2H₂O`
  `Fe²⁺ + NO + 5H₂O = [Fe(H₂O)₅(NO)]²⁺`
- **王水溶金**：
  `Au + HNO₃ + 4HCl = H[AuCl₄] + NO↑ + 2H₂O`（硝酸提供氧化性，氯离子提供配位能力降低金的电极电势）。

### 3. 竞赛易错点
- **易错1**：金属与硝酸反应生成H₂。**绝对错误**，硝酸与金属反应绝不生成氢气，而是生成氮的氧化物或铵盐。
- **易错2**：忽略NO₂与N₂O₄的平衡（`2NO₂ ⇌ N₂O₄`，降温有利于生成无色的N₂O₄，加压也会使颜色先变深后变浅）。
- **易错3**：混淆亚硝酸盐的配位方式（-NO₂ 硝基，-ONO 亚硝酸根），这在配合物异构体考题中是高频点。
