# 钛（Ti）与钒（V）- 竞赛核心考点

### 1. 核心形态（酸碱介质）
- **Ti(IV)**：**TiO²⁺ / Ti(OH)₄ / TiO₂**。最稳定氧化态。
  - 钛酰离子（TiO²⁺）：Ti(IV)在酸性水溶液中的主要存在形式，极易水解生成白色的偏钛酸（H₂TiO₃或TiO(OH)₂）沉淀。
- **Ti(III)**：**Ti³⁺**（紫红色）。强还原剂，在空气中易被氧化为Ti(IV)。
- **V(V)**：**VO₂⁺ / VO₄³⁻**。
  - 酸性中：**VO₂⁺**（淡黄色，二氧钒(V)离子），具中等氧化性。
  - 碱性中：**VO₄³⁻**（无色，正钒酸根），随pH降低会发生复杂的缩聚反应，生成多钒酸盐（如V₁₀O₂₈⁶⁻，橙红色）。
- **V(IV)**：**VO²⁺**（蓝色，氧钒(IV)离子）。最稳定的氧化态。
- **V(III)**：**V³⁺**（绿色）。
- **V(II)**：**V²⁺**（紫色）。强还原剂。

### 2. 特征反应
- **Ti(IV)的过氧化物配合物（鉴定Ti）**：
  在酸性条件下，TiO²⁺与H₂O₂反应生成橙黄色的过氧配合物。
  $$ \ce{TiO²⁺ + H₂O₂ + H₂O <->> [Ti(O₂)(OH)]⁺ + 2H⁺} $$（极灵敏，用于比色测定Ti或H₂O₂）。
- **钒的四种颜色变化（钒的还原）**：
  在酸性条件下，用锌锌汞齐还原V(V)，溶液颜色发生特征变化：
  **VO₂⁺（黄）** → **VO²⁺（蓝）** → **V³⁺（绿）** → **V²⁺（紫）**。
  $$ \ce{2VO₂⁺ + Zn + 4H⁺ -> 2VO²⁺ + Zn²⁺ + 2H₂O} $$
  $$ \ce{2VO²⁺ + Zn + 4H⁺ -> 2V³⁺ + Zn²⁺ + 2H₂O} $$
  $$ \ce{2V³⁺ + Zn -> 2V²⁺ + Zn²⁺} $$
- **钛的冶炼（Kroll法）**：
  $$ \ce{TiO₂ + 2C + 2Cl₂ -> TiCl₄ + 2CO} $$（高温，氯化）
  $$ \ce{TiCl₄ + 2Mg -> Ti + 2MgCl₂} $$（高温，Ar气保护下还原）

### 3. 竞赛易错点
- **易错1**：将Ti(IV)在水溶液中的形态写成Ti⁴⁺。**错误**，Ti⁴⁺电荷高、半径小，极化力极强，在水中会强烈水解，实际存在的是TiO²⁺（钛酰离子）。
- **易错2**：混淆VO₂⁺和VO²⁺。
  - **VO₂⁺**：V为+5价，淡黄色。
  - **VO²⁺**：V为+4价，蓝色。
- **易错3**：忽略钒酸盐的缩聚特性。与磷酸盐类似，钒酸盐在酸化时不会直接生成V₂O₅沉淀，而是先缩聚成多钒酸盐（颜色由无色变橙红），这是多酸化学的典型代表。
