# 碳（C）与硅（Si）、硼（B）- 竞赛核心考点

### 1. 核心形态（酸碱介质）
- **C(IV)**：**CO₂ / H₂CO₃ / CO₃²⁻ / HCO₃⁻**。
  - 碳酸是弱酸，CO₃²⁻易水解显碱性。
  - 许多金属离子（如Cu²⁺、Mg²⁺、Zn²⁺）与CO₃²⁻反应生成碱式碳酸盐沉淀。
- **C(II)**：**CO / CN⁻ / CNO⁻**。
  - CO是强还原剂（冶炼金属）和极强的配体（羰基配合物，如[Ni(CO)₄]）。
  - 氰根（CN⁻）是拟卤素，极毒，极强的配体（如[Fe(CN)₆]⁴⁻、[Ag(CN)₂]⁻），能溶解Au、Ag。
- **Si(IV)**：**SiO₂ / H₂SiO₃ / SiO₃²⁻**。
  - 硅酸是极弱酸（弱于碳酸），易形成胶体。
  - 硅酸盐结构极其复杂（多面体共顶点连接）。
- **B(III)**：**B₂O₃ / H₃BO₃ / BO₃³⁻ / B₄O₇²⁻**。
  - 硼酸（H₃BO₃）是**一元极弱酸**（缺电子，通过结合水电离出的OH⁻显酸性）。
  - 硼砂（Na₂B₄O₇·10H₂O）是重要的缓冲物质和基准试剂。

### 2. 特征反应
- **氰化法提金**：
  利用CN⁻的强配位能力，降低Au的电极电势，使其能被空气中的O₂氧化。
  `4Au + 8CN⁻ + O₂ + 2H₂O = 4[Au(CN)₂]⁻ + 4OH⁻`
- **硅酸盐的生成与溶解**：
  - 玻璃刻蚀：`SiO₂ + 4HF = SiF₄↑ + 2H₂O`（唯一能溶解SiO₂的酸）。
  - 强碱腐蚀：`SiO₂ + 2NaOH = Na₂SiO₃ + H₂O`（玻璃瓶不能盛放强碱）。
- **硼酸的酸性增强（与多羟基化合物配合）**：
  H₃BO₃本身极弱，不能直接用NaOH滴定。但加入甘油或甘露醇后，形成稳定的配合物，酸性显著增强，可用酚酞作指示剂进行滴定。
- **硼砂珠实验**：
  硼砂在高温下熔融生成玻璃状的偏硼酸钠和三氧化二硼（`Na₂B₄O₇ = 2NaBO₂ + B₂O₃`），B₂O₃能与许多金属氧化物结合生成特征颜色的偏硼酸盐（如Co(BO₂)₂呈蓝色）。

### 3. 竞赛易错点
- **易错1**：认为H₃BO₃是三元酸。**绝对错误**，H₃BO₃是缺电子分子，其电离方程式为：`H₃BO₃ + H₂O ⇌ [B(OH)₄]⁻ + H⁺`，因此是一元酸。
- **易错2**：混淆CO和CO₂的配位能力。CO有孤对电子和空的反馈π反键轨道，是极强的配体；CO₂没有孤对电子，极难作为配体。
- **易错3**：忽略硅酸盐的复杂结构。在书写硅酸盐化学式时，常写成氧化物形式（如正长石KAlSi₃O₈写为K₂O·Al₂O₃·6SiO₂），但必须清楚其真实结构是硅氧四面体骨架。
