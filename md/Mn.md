# 锰（Mn）- 竞赛核心考点

### 1. 核心形态（酸碱介质）
- **Mn(VII)**：**MnO₄⁻**（高锰酸根，紫红色）。在不同介质中还原产物不同：
  - **强酸性**：还原为 **Mn²⁺**（肉红色/近无色）。
  - **中性/弱碱性**：还原为 **MnO₂**（棕黑色沉淀）。
  - **强碱性**：还原为 **MnO₄²⁻**（锰酸根，绿色）。
- **Mn(VI)**：**MnO₄²⁻**（锰酸根，绿色），仅在强碱性中稳定，酸性或中性下极易歧化。
- **Mn(II)**：**Mn²⁺**（肉红色），在酸性中稳定，在碱性中生成白色的 **Mn(OH)₂** 沉淀，该沉淀在空气中迅速被氧化，变棕色最终成为 **MnO(OH)₂**（或写为MnO₂·xH₂O）。

### 2. 特征反应
- **Mn(VI)的歧化反应**：
  `3MnO₄²⁻ + 4H⁺ = 2MnO₄⁻（紫红） + MnO₂↓（棕黑） + 2H₂O`（锰酸根在酸性/中性下歧化）
- **Mn(II)的鉴定（氧化为高锰酸根）**：
  在酸性条件下，Mn²⁺可被强氧化剂（如NaBiO₃、PbO₂、(NH₄)₂S₂O₈/Ag⁺）氧化为紫红色的MnO₄⁻。
  `2Mn²⁺ + 5BiO₃⁻ + 14H⁺ = 2MnO₄⁻ + 5Bi³⁺ + 7H₂O`
- **Mn(OH)₂的氧化**：
  `2Mn(OH)₂（白） + O₂ = 2MnO(OH)₂（棕黑）`

### 3. 竞赛易错点
- **易错1**：高锰酸钾滴定配平错误。酸性介质必须用H₂SO₄，不能用HCl（会被KMnO₄氧化产生Cl₂）或HNO₃（本身有强氧化性干扰）。
- **易错2**：误认为MnO₄⁻在任何条件下都还原为Mn²⁺。必须牢记“酸锰二、中锰四、碱锰六”的规律。
- **易错3**：写Mn(II)与碱反应时，忽略空气中氧气的参与，直接写成稳定的Mn(OH)₂（实际极易变质）。
