# 卤素（Cl, Br, I）- 竞赛核心考点

### 1. 核心形态（酸碱介质）
- **X(VII)**：**ClO₄⁻ / BrO₄⁻ / IO₄⁻**。
  - 高氯酸（HClO₄）是最强无机含氧酸，ClO₄⁻在溶液中动力学极稳定，常作惰性阴离子。
  - 高溴酸（HBrO₄）极不稳定，是极强的氧化剂。
  - 高碘酸（H₅IO₆ / IO₄⁻）在酸性中是强氧化剂，能将Mn²⁺氧化为MnO₄⁻，且能裂解邻二醇。
- **X(V)**：**ClO₃⁻ / BrO₃⁻ / IO₃⁻**。
  - 氯酸盐和溴酸盐在酸性下具强氧化性。
  - 碘酸盐（IO₃⁻）相对稳定，常用于滴定分析（如与I⁻发生归中反应）。
- **X(I)**：**ClO⁻ / BrO⁻ / IO⁻**。
  - 仅在碱性中存在（次卤酸盐）。在酸性中极易歧化或分解。
  - 稳定性：ClO⁻ > BrO⁻ > IO⁻。IO⁻在室温下即迅速歧化为IO₃⁻和I⁻。

### 2. 特征反应
- **卤素单质的歧化（碱性介质）**：
  - 冷稀碱：`X₂ + 2OH⁻ = X⁻ + XO⁻ + H₂O`（Cl₂, Br₂适用，I₂极快转化为IO₃⁻）。
  - 热浓碱：`3X₂ + 6OH⁻ = 5X⁻ + XO₃⁻ + 3H₂O`（Cl₂, Br₂, I₂均适用）。
- **碘酸根与碘离子的归中（酸性介质）**：
  `IO₃⁻ + 5I⁻ + 6H⁺ = 3I₂ + 3H₂O`（定量分析中极重要的反应，加碱则逆向歧化）。
- **拟卤素反应（以CN⁻、SCN⁻为例）**：
  性质类似卤素，如 `(CN)₂ + 2OH⁻ = CN⁻ + CNO⁻ + H₂O`。

### 3. 竞赛易错点
- **易错1**：认为F₂能从水溶液中置换出Cl₂。**绝对错误**，F₂会直接与水剧烈反应：`2F₂ + 2H₂O = 4HF + O₂`。
- **易错2**：忽略I₂在KI溶液中的溶解度增大原理。`I₂ + I⁻ ⇌ I₃⁻`（形成多卤化物，棕黄色溶液）。
- **易错3**：配平次氯酸盐氧化反应时介质错误。ClO⁻只能在碱性或中性条件下存在，若在酸性条件下，应写为HClO，且极易放出Cl₂（`ClO⁻ + Cl⁻ + 2H⁺ = Cl₂↑ + H₂O`）。
