# 化学键理论

## 价层电子对互斥理论 (VSEPR Theory)
用于预测分子的几何构型。中心原子的价层电子对（成键对+孤对）倾向于尽可能远离，以减小排斥力。

### 构型判断步骤
1.  计算价层电子对数 = (中心原子价电子数 + 配位原子提供电子数 - 电荷) / 2。
2.  确定电子对几何构型 (直线, 平面三角形, 四面体, 三角双锥, 八面体)。
3.  考虑孤对电子 (LP) 对成键电子 (BP) 的排斥：LP-LP > LP-BP > BP-BP。

### 常见构型
*   **AX2**: 直线形 ($180^\circ$), e.g., $BeCl_2$, $CO_2$
*   **AX3**: 平面三角形 ($120^\circ$), e.g., $BF_3$
*   **AX2E1**: V形 (<$120^\circ$), e.g., $SO_2$
*   **AX4**: 正四面体 ($109.5^\circ$), e.g., $CH_4$
*   **AX3E1**: 三角锥 (<$109.5^\circ$), e.g., $NH_3$
*   **AX2E2**: V形 (<$109.5^\circ$), e.g., $H_2O$
*   **AX5**: 三角双锥 ($90^\circ, 120^\circ$), e.g., $PCl_5$
    *   *Berry假旋转*: 轴向与赤道向配体的快速交换。
*   **AX4E1**: 变形四面体 (跷跷板), e.g., $SF_4$
*   **AX3E2**: T形, e.g., $ClF_3$
*   **AX2E3**: 直线形, e.g., $XeF_2$
*   **AX6**: 正八面体 ($90^\circ$), e.g., $SF_6$
*   **AX5E1**: 四方锥, e.g., $BrF_5$
*   **AX4E2**: 平面四方形, e.g., $XeF_4$

## 价键理论 (Valence Bond Theory)

### 杂化轨道理论 (Hybridization)
原子轨道混合形成新的简并杂化轨道，用于成键。
*   **sp**: 1s + 1p，直线形。
*   **sp2**: 1s + 2p，平面三角形。
*   **sp3**: 1s + 3p，四面体。
*   **dsp2**: 1d + 1s + 2p，平面四方形（常见于过渡金属配合物）。
*   **sp3d**: 1s + 3p + 1d，三角双锥。
*   **sp3d2**: 1s + 3p + 2d，八面体。

## 分子轨道理论 (Molecular Orbital Theory)
原子轨道 (AO) 线性组合形成分子轨道 (MO)。
$$ \Psi = c_1\phi_1 \pm c_2\phi_2 $$

### 双原子分子 MO 能级图
1.  **同核双原子分子 ($O_2, F_2$)**:
    *   $\sigma_{1s} < \sigma^*_{1s} < \sigma_{2s} < \sigma^*_{2s} < \sigma_{2p_z} < \pi_{2p} < \pi^*_{2p} < \sigma^*_{2p_z}$
2.  **同核双原子分子 ($B_2, C_2, N_2$)**:
    *   由于 s-p 轨道混合严重，$\sigma_{2p_z}$ 能量升高超过 $\pi_{2p}$。
    *   $\sigma_{1s} < \sigma^*_{1s} < \sigma_{2s} < \sigma^*_{2s} < \pi_{2p} < \sigma_{2p_z} < \pi^*_{2p} < \sigma^*_{2p_z}$
3.  **异核双原子分子 (如 $CO, NO$)**:
    *   电负性大的原子轨道能量更低，成键轨道更多成分来自电负性大的原子。

### 键级与磁性
*   **键级 (Bond Order)** = (成键电子数 - 反键电子数) / 2。键级越大，键越短，键能越大。
*   **磁性**: 若分子轨道中有未成对电子，分子显**顺磁性** (Paramagnetic)；若所有电子配对，显**反磁性** (Diamagnetic)。
    *   例如：$O_2$ 有两个未成对电子在 $\pi^*$ 轨道，故为顺磁性。

## 离域 $\pi$ 键 (Delocalized Pi Bonds)
当分子的骨架由 $\sigma$ 键构成，且多个相邻原子提供平行的 p 轨道时，形成离域 $\pi$ 键。
*   **大 $\pi$ 键符号**: $\Pi_n^m$ (n个原子，m个电子)。
*   **例子**:
    *   苯 ($C_6H_6$): $\Pi_6^6$。
    *   碳酸根 ($CO_3^{2-}$): $\Pi_4^6$ (中心 C 提供空 p 轨道或 1 个电子，视共振结构而定，实际为 4 轨道 6 电子)。
    *   臭氧 ($O_3$): $\Pi_3^4$。
*   **共轭效应**: 使体系能量降低，增加稳定性。
