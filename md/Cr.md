# 铬（Cr）- 竞赛核心考点

### 1. 核心形态（酸碱介质）
- **Cr(VI)**：酸性介质中主要为 **Cr₂O₇²⁻**（重铬酸根，橙红色）；碱性介质中主要为 **CrO₄²⁻**（铬酸根，黄色）。两者存在平衡：$$ \ce{2CrO₄²⁻ + 2H⁺ <->> Cr₂O₇²⁻ + H₂O} $$（化合价不变，仅形态/颜色变）。
- **Cr(III)**：两性离子。酸性中为 **Cr³⁺**（蓝绿色/绿色，水合离子为紫色但常呈绿色）；碱性中为 **[Cr(OH)₄]⁻**（亮绿色，或写为CrO₂⁻）。加碱先生成灰绿色沉淀 **Cr(OH)₃**，碱过量则溶解。
- **Cr(II)**：**Cr²⁺**（亮蓝色），极强的还原剂，在空气中极易被氧化为Cr³⁺。

### 2. 特征反应
- **Cr(VI)的氧化性**：在酸性条件下，K₂Cr₂O₇ 是强氧化剂。
  $$ \ce{Cr₂O₇²⁻ + 6Fe²⁺ + 14H⁺ -> 2Cr³⁺ + 6Fe³⁺ + 7H₂O} $$（橙红色褪去，变为绿色）
- **过氧化铬的生成（鉴定Cr⁶⁺的特征反应）**：
  在酸性Cr₂O₇²⁻溶液中加入H₂O₂和乙醚，生成深蓝色的 **CrO(O₂)₂**（或写为CrO₅，蝴蝶型结构），萃取到乙醚层中变稳定。
  $$ \ce{Cr₂O₇²⁻ + 4H₂O₂ + 2H⁺ -> 2CrO₅ + 5H₂O} $$
- **Cr(III)的氧化**：在碱性条件下，Cr(III)极易被H₂O₂、卤素等氧化为Cr(VI)。
  $$ \ce{2[Cr(OH)₄]⁻ + 3H₂O₂ + 2OH⁻ -> 2CrO₄²⁻ + 8H₂O} $$（亮绿色变为黄色）

### 3. 竞赛易错点
- **易错1**：把Cr⁶⁺碱性形态写成Cr₂O₇²⁻（正确为CrO₄²⁻，碱性环境下Cr₂O₇²⁻会转化为CrO₄²⁻）。
- $$ \ce{**易错2**：忽略CrO₅的结构和化合价。CrO₅中Cr为+6价，包含两个过氧键（-O-O-）和一个Cr->O双键。} $$
- **易错3**：配平Cr(III)在碱性条件下的氧化还原方程式时，产物错写为Cr³⁺或Cr₂O₇²⁻（碱性应为CrO₄²⁻）。
