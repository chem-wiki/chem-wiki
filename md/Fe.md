# 铁（Fe）- 竞赛核心考点

### 1. 核心形态（酸碱介质）
- **Fe(VI)**：**FeO₄²⁻**（高铁酸根，紫红色）。仅在强碱性环境中稳定，极强的氧化剂（氧化性强于KMnO₄），在酸性中迅速分解放出O₂。
- **Fe(III)**：**Fe³⁺**（淡紫色，水解常呈黄色/棕黄色）。易水解，酸性中稳定；碱性中生成红褐色 **Fe(OH)₃** 沉淀。具有中等强度的氧化性。
- **Fe(II)**：**Fe²⁺**（浅绿色）。还原性较强，在空气中或碱性条件下极易被氧化为Fe(III)。碱性中生成白色 **Fe(OH)₂** 沉淀，迅速变灰绿，最终变红褐。

### 2. 特征反应
- **Fe(III)的特征鉴别**：
  - 与SCN⁻：`Fe³⁺ + nSCN⁻ ⇌ [Fe(SCN)n]³⁻ⁿ`（血红色溶液，n=1~6）。
  - 与黄血盐（K₄[Fe(CN)₆]）：生成普鲁士蓝沉淀 `K⁺ + Fe³⁺ + [Fe(CN)₆]⁴⁻ = KFe[Fe(CN)₆]↓`。
- **Fe(II)的特征鉴别**：
  - 与赤血盐（K₃[Fe(CN)₆]）：生成滕氏蓝沉淀（现证实与普鲁士蓝结构相同） `K⁺ + Fe²⁺ + [Fe(CN)₆]³⁻ = KFe[Fe(CN)₆]↓`。
  - 与邻二氮菲（phen）：生成橘红色配合物 `[Fe(phen)₃]²⁺`（常用于定量比色测定）。
- **高铁酸钾的制备（碱性氧化）**：
  `2Fe(OH)₃ + 3ClO⁻ + 4OH⁻ = 2FeO₄²⁻ + 3Cl⁻ + 5H₂O`

### 3. 竞赛易错点
- **易错1**：认为Fe³⁺与I⁻可以共存。实际上会发生氧化还原：`2Fe³⁺ + 2I⁻ = 2Fe²⁺ + I₂`。
- **易错2**：配制Fe²⁺溶液时不加酸和铁钉。必须加酸抑制水解，加铁粉/铁钉防止被空气氧化（`2Fe³⁺ + Fe = 3Fe²⁺`）。
- **易错3**：忽略Fe(III)的强水解性。将FeCl₃溶液蒸干灼烧，得到的产物是Fe₂O₃而不是FeCl₃（`FeCl₃ + 3H₂O ⇌ Fe(OH)₃ + 3HCl↑`，加热促进HCl挥发）。
